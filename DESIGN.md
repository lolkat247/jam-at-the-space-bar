# Design

## Format

- 2D pixel art, 32×32 tile/sprite size

## Core Loop

1 ...
3. **Brew** – return to the bar and combine fruits into a jar via a rhythm minigame
4. **Serve** – serve the result to customers and receive a rating based on the brew

## Scenes

### Overworld (Foraging)
- One scene per fruit type
  - Passed Via props
- Fruits must be caught via the overworld minigame


### Bar
- Home base where brewing and serving happen
- Player combines collected fruits into jars
- Customers are served here and rate the result
