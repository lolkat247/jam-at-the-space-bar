# Jam at the Space Bar

"A short description of the project."

## Requirements

- [Godot 4.6.1](https://godotengine.org/download)

## Setup

1. Clone the repo
   ```bash
   git clone https://github.com/your-username/your-repo.git
   ```
2. Open Godot and click **Import**, then select the `project.godot` file
3. Let Godot reimport assets, then hit **Play**

## Notes

The `.godot/` folder is gitignored and will be generated locally on first open.

Branches are for chumps. Don't use them for this jam!!!
Please pull often.
